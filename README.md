# 🎮 Projeto X - Número Secreto

Um jogo onde o objetivo é adivinhar um número secreto gerado aleatoriamente entre **1 e 70**.

A cada tentativa, o jogo informa se o número secreto é **maior** ou **menor** que o palpite. Ao acertar, é exibida uma mensagem de vitória com o número de tentativas e o jogador pode iniciar uma nova partida.
